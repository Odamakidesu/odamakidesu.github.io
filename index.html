File Mover App

このページは、Google OAuth 審査のためのアプリ紹介ページです。

アプリについて

File Mover Appは、GoogleカレンダーやGoogle Driveと連携して、画像・ZIPファイルの整理や予定取得などを自動化するユーティリティアプリです。

本アプリは限定配布型であり、Google認証を通してのみ使用可能です。
